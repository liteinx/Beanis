🥂 Beanis

APK Distribution System

📱 What is Beanis?

Beanis is a bash-based APK distribution system for sharing Android APK files securely. It uses a template-based approach to manage multiple APK variants for different clients while maintaining consistent metadata.

✨ Features

  • Template-based APK management
  • Client-specific download pages with secure tokens
  • Minimalistic, modern web interface
  • Automatic APK metadata tracking (SHA256, file size, etc.)
  • Static site generation - no database required

🚀 Quick Start

# Create an APK template
./scripts/create_template.sh

# Add APKs for different clients
./scripts/add_apk.sh /path/to/app.apk --template duckduckgo --client client1

# Generate the distribution website
./scripts/generate_site.sh

# Serve the site
cd public && python3 -m http.server 8080
